## 网站监测（用户数 / 停留时长 / 行为分析）

项目支持接入 **Google Analytics 4** 与 **百度统计**，用于查看用户数、停留时长、用户行为等，便于后续优化。无需自建后台，配置环境变量即可。详见 [ANALYTICS.md](./ANALYTICS.md)。

## Install packages

Install packages in Shell pane or search and add in Packages pane.
